# Employee Attrition Prediction ML Model 

# Overview

This project is a Machine Learning pipeline built to predict employee attrition using Python and Scikit-learn. The model analyzes employee-related factors and predicts whether an employee is likely to leave the company.

The project demonstrates:
Data preprocessing
Feature engineering
Machine Learning pipelines
Model training and evaluation
Employee attrition prediction using classification algorithms

# Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

# Project Workflow
- Load Employee Attrition Dataset
- Data Cleaning & Preprocessing
- Label Encoding
- Train-Test Split
- Pipeline Creation
- Model Training using Random Forest Classifier
- Model Evaluation
- Prediction Generation

# Machine Learning Pipeline
The project uses:
- ColumnTransformer
- Pipeline
- StandardScaler
- OneHotEncoder
- RandomForestClassifier

# Model Performance

The model performance is evaluated using:
- Accuracy Score
- Classification Report
